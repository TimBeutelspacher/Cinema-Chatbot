# Cinema Chatbot
### Prüfungsleistung für Knowledge Representation and Natural Language Processing
#### Sommersemester 22

Dieses Repository enthält den Code eines prototypischen Chatbots für die Website eines Kinos.
Der CinemaBot übernimmt folgende Aufgaben:
- Buchung von Tickets
  - Validierung der Eigabedaten
  - Korrekter Umgang mit Unterbrechungen der Form
- Fragen eines FAQs beantworten

Das GUI des Chatbots wurde mit der JavaScript Library React umgesetzt.
